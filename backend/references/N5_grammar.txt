【N5文法・重要項目と対策】

### 重要な文法項目
1. **助詞（は・が・を・に・で・へ）**
   - 私は学生（は / です）。=> 私は学生です。
   - 私はりんご（を）食べます。
   - 図書館（で）勉強します。
   - 学校（へ / に）行きます。

2. **時間・場所の表現**
   - 7時（に）起きます。
   - 机の上（に）本があります。

3. **動詞の活用（ます形・て形・ない形・辞書形）**
   - 飲みます / 飲んで / 飲まない / 飲む
   - 食べます / 食べて / 食べない / 食べる
   - します / して / しない / する
   - 来ます（きます） / 来て（きて） / 来ない（こない） / 来る（くる）

4. **形容詞（い形容詞・な形容詞）**
   - [い] 大きいですか。 -> いいえ、大きくないです。
   - [な] 好きですか。 -> いいえ、好きじゃありません。

5. **〜たいです / 〜ましょう / 〜てください**
   - 水を（飲みたいです）。
   - 一緒に（帰りましょう）。
   - ここに名前を（書いてください）。

### 過去問風サンプル問題
（AIはこのスタイルを真似してください）

問題：( )に何を入れますか。1・2・3・4から一番いいものを一つ選んでください。

1. 私は　毎日　コーヒーを　（　　　）。
   1. 飲みます
   2. 飲みました
   3. 飲む
   4. 飲んで
   正解: 1 (Habitual action)

2. あした、一緒に　映画を　（　　　）か。
   1. 見ます
   2. 見ません
   3. 見ましょう
   4. 見てください
   正解: 2 ("Won't you...?" invitation - mïmasenka)

3. 机の　上に　本が　（　　　）。
   1. あります
   2. います
   3. ある
   4. いる
   正解: 1 (Inanimate object existence)

4. 図書館で　本を　（　　　）　いけません。
   1. 読んで
   2. 読んでは
   3. 読んでも
   4. 読むと
   正解: 2 (Must not - te wa ikemasen)

5. この　りんごは　（　　　）　ないです。
   1. 甘く
   2. 甘い
   3. 甘くて
   4. 甘
   正解: 1 (i-adjective negation: amaku-nai)
